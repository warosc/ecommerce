# Optimus Engineering Kit

## Visión
Construir una plataforma open source para Óptica Optimus, completamente dockerizada, basada en microservicios, DDD, Clean Architecture y preparada para Kubernetes.

## Reglas para Claude Code
- Nunca copiar código, diseños o contenido de terceros.
- Trabajar por fases.
- Cada fase debe incluir documentación, pruebas y Docker.
- Mantener cobertura >90%.
- Generar ADR cuando haya decisiones arquitectónicas.

## Stack
Next.js, NestJS, PostgreSQL, Redis, RabbitMQ, MinIO, OpenSearch, Keycloak, Traefik.

## Módulos
Landing, Ecommerce, Catálogo, Inventario, POS, CRM, Agenda, Pacientes, Facturación, Marketing, Analytics, Admin.

## IA (opcional)
Repositorio independiente optimus-ai con Ollama y OpenWebUI. El Virtual Try-On usa principalmente MediaPipe + Three.js + OpenCV.
